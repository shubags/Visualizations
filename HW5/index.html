Project Description

The purpose of this assignment was to:

  • Practice the creation of a maps using TopoJSON

Functionality was built in phases to successively build on features. The following phases of describe the features added.

Basic functionality (C-Level)

  • Modified the original code to show circles where earthquakes hit
  • Modified code to make circle radius' the magnitude of the earthquakes
  • Changed the message at the bottom of the visualization to read what the magnitude of the earthquake was and the lat/long of the event

Design decisions

I chose to truncate the map of the world to ensure that the user could read the text below the visualization. By trying to fit the entire map, the user would have had to put the curson on the circle at a point where they would need to scroll down to read the event details (which would not work). I chose to keep the mercator projection because it is easy for people to interpret because it keeps everything in rectangular scales. Many of the issues related to mercator projections were mitigated by the truncation of high longitude locations (i.e. everything near the poles).
