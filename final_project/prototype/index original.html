Project Description

The purpose of this assignment was to:

  • Practice display of time-series data

Functionality was built in phases to successively build on features. The following phases of describe the features added.

Basic functionality (C-Level)

  • Loaded data from externally processed .csv file
  • Drew individual line charts for the total deaths and serious injuries based on the seat that the individual was sitting in in the car
  • Added text to the end of the lines to indicate which data set the line was associated with
  • Drew stacked area charts for the total deaths and serious injuries based on the seat that the individual was sitting in in the car
  • Drew color box legend to indicate which data that the area chart represented

Moderate functionality (B-Level)

  • Added brushing to a third plot that allows focus and context for a more specific measurement of driving death rates.

Design decisions

The design decisions of this chart can be broken down into two primary categories: aesthetics and data representation. These factors were meant to improve the understandability of the data.

From an data representation perspective, I first wanted to clearly show the magnitude difference that each seat had with respect to deaths and serious injuries. I also wanted the reader to see how the injuries and deaths followed similar patterns with respect to time, irrespective of where in the car the individual was sitting. This was accomplished by the multiple-line chart. In order to show the overall total of injuries and deaths, I created the stacked area chart. Granularity was added to show the proportion of deaths/injuries that were suffered from each car seat location.

From an aesthetics perspective, I allowed d3 to select the best three colors for the car seat locations.
