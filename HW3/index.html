Project Description

The purpose of this assignment was to:

  • Practice display of multi-variate data
  • Exploring the capabilities of d3 and small-multiple plots

Functionality was built in phases to successively build on features. The following phases of describe the features added.

Basic functionality (C-Level)

  • Loaded data from externally processed .csv file
  • Built scatter plot with graduation and illiteracy rates
  • Modified radius to scatter plot circles based on murder rate
  • Modified color of the scatter plot based on the region that the state was in

Design decisions

The design decisions of this chart can be broken down into two primary categories: aesthetics and data representation. These factors were meant to improve the understandability of the data.

From an data representation perspective, I first wanted the easiest relationship to be seen to be the linear relationship between high school graduation rate and illiteracy. After this, I wanted to couple the education level with the notion of safety. I used bubble size in order to represent this. Lastly, I wanted to show how states in certain regions were clustered for these three factors, which I represented with the colors.

From an aesthetics perspective, I allowed d3 to select the best four colors for my regions. I added some opacity to improve the ability to see overlapping circles. Despite the presence of axes and labels, I added a tool tip feature so that could easily see all of the exact values of a state by hovering. This is a form of double encoding, but it was meant to help users who find it difficult to measure the area of the bubbles.
